# NovaTeam - Portfolio

Portfolio simple y moderno para NovaTeam, negocio de desarrollo de páginas web y sistemas.

## Colores

- **Azul Oscuro**: #051839
- **Azul Claro**: #20a1e4
- **Blanco**: #ffffff

## Estructura

- **Hero Section**: Presentación de NovaTeam con slogan
- **Muestras**: 3 espacios para previsualizaciones de webs
- **Proyectos**: 1 espacio para proyecto realizado con cliente
- **Contacto**: Formulario de contacto al final

## Uso

### Desarrollo Local
Simplemente abre `index.html` en tu navegador o despliega en tu servidor preferido.

### GitHub Pages
Este proyecto está configurado para funcionar con GitHub Pages. Las rutas no incluyen la extensión `.html` para URLs más limpias.

1. Sube el proyecto a un repositorio de GitHub
2. Ve a Settings > Pages
3. Selecciona la rama principal como fuente
4. El sitio estará disponible en `https://tuusuario.github.io/nombre-repositorio/`

## Personalización

- Agrega imágenes a las muestras reemplazando los placeholders
- Completa la información del proyecto en la sección correspondiente
- Configura el envío del formulario de contacto según tus necesidades
